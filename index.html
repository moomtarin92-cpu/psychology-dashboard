Google Apps Script URL

⚠️ Important: You must also update your Google Apps Script to the new version below. The old script uses doPost() for writes, which causes CORS errors from browsers. The new script handles everything via doGet() with URL parameters — this is the correct way to call Apps Script from a static website.

📋 Apps Script Setup — Step by Step

1
Open your Google Sheet → go to Extensions → Apps Script
2
Make sure your sheet has two tabs named exactly: Patients and Sessions
3
Delete all existing code in the script editor and paste the new code below. Click Save (Ctrl+S).
4
Click Deploy → New Deployment. Type: Web App. Execute as: Me. Who has access: Anyone. Click Deploy and authorize. Copy the URL.
5
Paste the URL above and click Save & Test. You should see ✅ Connected.
