📋 Google Apps Script Setup Guide

Step 1: Open Google Sheets and create a new spreadsheet.

Step 2: Create two sheets named exactly: Patients and Sessions

Step 3: Go to Extensions → Apps Script, paste the code below, and save.

Step 4: Click Deploy → New Deployment → Web App. Set access to "Anyone". Copy the URL and paste it above.
